# Decision Support Research

Research for a real decision, not for background reading.

Read shared references:
- `../researcher/references/core-method.md`
- `../researcher/references/source-hierarchy.md`
- `../researcher/references/frontier-management.md`
- purpose-specific source packs from `../researcher/references/` as needed

## Workflow

1. Define the decision, options, constraints, and time horizon.
2. Build evaluation criteria that actually matter to the choice.
3. Gather evidence by criterion, not by random topic branch.
4. Stress-test downside, regret, and hidden costs.
5. Recommend a path or reduce the option set with explicit confidence.

## Constraints

- Optimize for decision relevance, not maximum information volume.
- Make tradeoffs legible.
- Say what could change the recommendation.

## Output

Create or update `research-{topic}.md` while working so criteria, tradeoffs,
and recommendation logic stay externalized.

Prefer sections like:
- Decision Frame
- Options And Criteria
- Evidence By Criterion
- Tradeoffs
- Recommendation
- What Would Change The Call
